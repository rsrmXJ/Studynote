在今天这个人才竞争日益激烈的时代，职业生涯规划在就业中的重要性。当代大学生若是一脸茫然的进入这个竞争激烈的社会，怎能使自己占有一席之地。因此，我们拟定一份职业生涯规划，就相当于找到了一条适合和属于自己的路，也就摆脱了迷茫。

职业生涯活动将伴随我们的大半生，只有拥有适合的职业生涯规划才更有可能实现完美的人生，因此职业生涯规划具有特别重要的意义。

我希望通过生涯规划可以更好的认识自我，学到更多，并充实自己，努力规划好未来的人生轨迹，用更优的计划，规避风险，走更优的路。

**我的职业生涯规划如下**：

### 一. 自我认知：

##### 1. 个人基本情况

我今年 19 岁，目前正在河南工业职业技术学院读书，成绩一般。我内心热爱生活，对未来充满希望。

##### **2. 兴趣分析**：

阅读：喜欢有目的阅读一些书籍。以解决在生活、学习等各方面遇到的问题。 在知识大爆炸的当今社会，眼光和格局决定了发展前景，而阅读是保证思想长青，不落后于时代的重要途径。  凡阅读皆有其目的。唯有明确阅读的目的，才有利于更加高效地阅读。  笛卡尔曾说：" 遍读好书，有如走访著书的前代高贤，同他们促膝谈心。而且是一种精湛的交谈，古人向我们谈出的只是他们最精粹的思想。书籍是前代高贤思想和智慧的结晶，他们是那个时代最具有智慧的人。经常说："再天才的发明和创造最初也是从模仿开始的"。而我们阅读他们的书籍，就是向那个时代最具有智慧的人学习,从而站在巨人的肩膀上。

编程：学习一种技能为未来的生活和工作作准备，而编程搞好可以满足需要(有需求才有动力)，对自己赖以谋生的技能必须精通。

其它兴趣：看小说、听音乐，健身；

##### **3. 性格分析**：

我是一个事业心强，注重个性发展的人。我看重的是是否能最大限度地发挥自己、是否具有独立性、是否能发挥自己的潜力，施展自己的本领，而收入地位及他人对自己的看法次之。

大多数时候，我沉浸在平凡和享乐之中，我想要改变，并且我也为之努力。

##### **4. 技能分析**：

- 在网络上搜索资源的能力和自学能力。

- 沟通技能：能够清楚、准确的表达自己的想法
- 解决问题或批判性思维技能。思考和沟通，最重要的不是为了说服他人，而是解决问题
- 专业技能：会用 Markdown、LaTeX 作一些笔记和 Git (版本控制)、Java

##### 5. 价值观：

一个人的首要问题是生存，生存是最大的目标。脱离物质谈理想是不现实的；对于一门职业，我最看重的不仅仅是物质的回报，也希望更多的是精神上的满足，而我愿我将来从事的职业可以实现双赢。兼顾现实与理想。于工作上勤勤恳恳，脚踏实地，与时俱进，实现我看重的人生价值和社会价值的双赢，成就自我，也奉献社会。

### 二. 职业认知

##### 1. 目标职业：

根据自己的职业兴趣和个人能力，我希望自己最终成为一名 Java 全栈工程师或网络爬虫工程师。

##### 2. 影响职业选择及相关外部环境分析

**(1) 家庭环境分析**

我家庭经济条件困难，家庭生活和谐。父母充分尊重我的意愿，期望我实现我的追求，从小到大，我接受更多的是来自他们的鼓励，而非压力。对于我想要成为 Java 全栈工程师他们也很支持。

**(2)社会环境分析**

当今社会，就业形势严峻，且对应聘者的学历和技术不断提高。而我的竞争对手，不仅仅是同龄的毕业生，还有许多比我经验丰富且学历高的人。

**(3)职业环境分析**

随着人工智能，云计算，物联网工程，计算机科学与技术，自动化，工业 4.0 的发展，计算机相关专业的毕业生就业前景还不错，发展潜力巨大。我国重视计算机等高新技术产业的发展。

##### **3.** 对职业的认知

**Java 全栈工程师**

java 全栈工程师一般指的是 JavaWEB 开发后台与前端开发，能够用 Java 编程独立完成完整的网站过程(开发成本降低)。

为什么我要选择这个职业？

- 一. 我已经学习了 Java 这门编程语言。每天都在解决 bug 或从而遇到的难题，是一种思维锻炼和解决问题能力的提升。简而言之，这既是我的优势，也是我的兴趣所在。
- 二. 定我未来的可能生活在二线及以下的城市。在二线及以下的城市，为了减少成本，岗位分工并步明细。公司会要技术方面全方面的人才从而降低成本。
- 三. 进一步可以继续深入的学习后台开发，退一步可以通过学习网络爬虫等技术，进入大数据领域，时刻跟紧最新东西

### 三. 目标决策

一个有理想的人，必须要有目标，不同的阶段，不同的目标。一个人一旦拥有了目标，就不会迷茫，也就找到了适合自己的路。

##### 1. 近期目标

经过系统的学习最终成为一名 Java 全栈工程师或网络爬虫工程师

##### 2. 长期目标

由于该职业的发展，需要考虑转行或创业。

### 四. 计划实施

##### 1. 大学期间

- 培养分析解决问题的能力，以及良好的实践技能
- 努力学好专业知识,增强独立思考能力；

- 参加一次竞赛，锻炼公众场合下的语言表达能力和抗压能力。此外，在提高专业能力的同时，我需要不断增强自身的人文素养，广泛涉猎人文哲学书籍。

- 坚持锻炼身体，为未来工作作准备
- 增强数学和英语的听说读写能力，因为软件在后期靠的就是数学和英语能力

##### 2. 毕业后 5 年

- 学习自己能够在工作中用到知识，有需求才用动力，在工作中扩大自己的职责范围，从而学习更多知识和经验,对自己赖以谋生的技能必须精通
- 注重向他人学习，成为一名技术过硬，能力突出的工程师
- 在闲暇时间学习一些自己感兴趣的东西
- 自考本科或专升本提升学历

##### 4. 毕业后 5~10 年

- 扩大职责范围，提升个人能力，时刻注重自我价值的提升
- 时刻紧跟追技术领域最新动向，时刻学习最新的技术
- 努力成为某一领域内的资深技术专家或转向管理岗位
- 如果机会合适可以考虑创业

### 五. 评估调整

##### 1. 调整时间

在一般状况下，定期（半年或一年）评估规划。总结不足和完成的较好的地方，将大计划化小，小化细。当遇到特殊情况时，随时进行评估以便及时对自己的规划做出调整。

##### 2. 职业目标评估

在今天这个巨变的时代，跟不上时代是最致命的挑战。多少坚固若长城的东西，迅速销声匿迹。变化是表，不变是里，在所有变化中，总有一些东西会天荒地老，历久弥新。 学如逆水行舟，不进则退，没有什么永不过时，只有不断学习和进步才能不落后。如果真的到了那个时候，我抛弃我的所学，对全新的领域深入学习

##### 3. 职业路径评估

我的职业发展路径 Java 后端开发工程师 >> Java 全栈开发工程师 >> 资深技术专家/管理岗位 >> 创业。在这个时代，唯一不变的就是变化。如果这个领域不行了，我将及时调整路径和发展方向。
